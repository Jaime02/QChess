# QChess

QChess is a minimalist chess game made by four students using Python 3 and PyQt5.

# Installation:

 - Requierements: Python 3.6 or greater
 1. Install PyQt5
 
    ```shell
    pip install PyQt5
    ```
    or
    ```shell
    pip install -r requierements.txt
    ```
2. Run the program
```shell
python main.py
```
